# Restaurant POS Billing System

A simple Point of Sale (POS) Billing System developed using HTML, CSS, Bootstrap, and JavaScript. The application allows users to select product quantities, calculate totals automatically, apply discounts, and generate a dynamic receipt.

## Features

* Product quantity management
* Automatic price calculation
* Grand total calculation
* Discount calculation
* Final payment calculation
* Dynamic receipt generation
* Responsive user interface
* Bootstrap-based design

## Technologies Used

* HTML5
* CSS3
* Bootstrap 5
* JavaScript

## Project Overview

This project simulates a basic restaurant billing system where users can enter product quantities and instantly view calculated totals. The system automatically applies discounts and generates a receipt showing selected items and billing details.
